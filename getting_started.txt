####
# Installing Python
####

Navigate to http://www.python.org/getit and download the Python 2.7.5 (32bit) installer for your system.


####
# Installing a nicer editor (OPTIONAL)
####

Go to http://www.sublimetext.com/ and install the SublimeText (it's free)...
